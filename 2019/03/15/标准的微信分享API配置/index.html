标准的微信分享API配置
发布于：2019-03-15 13:16:20 标签：/ 微信 / 前端 / 访问：

最开始在老杨那里就遇到了这个问题，微信分享，后来同事帮忙搞了出来，不过需要通过微信桌面客户端来实现，这一次深入接触发现我们只是使用了接口的擦边球。。。本次详细记录微信分享自定义接口的使用。

首先需要域名、服务器以及一个认证公众号，这就算是齐活了。

接下来就是配置接口，这里我们接触到了AppID，AppSecret，域名白名单以及IP白名单，我们一步步来做：

  1. 添加JS接口安全域名

    点击设置进入设置界面添加，每个公众号只能添加3个域名(经过ICP备案、每月保存/修改次数3次)并且下载验证文件传至域名所在服务器的根目录，完成第一步配置。

  2. 保存AppID与AppSecret，添加IP白名单

    在公众号的开发→基本配置中保存AppID并重置并保存开发者密码

    点击IP白名单的查看添加域名所在的IP地址，一路下来，微信分享接口的基本配置已完成。
